State Machine Editor

An interactive editor for building state machine diagrams for students.

×

I created a visual editor in JavaFX to demonstrate my skills with immediate-mode graphics, interaction with graphics, more complex model-view-controller architectures, and multiple synchronized views. The system is an interactive editor for building state-machine diagrams. The editor allows users to select a tool from a tool palette. Different tools allow the user to create state nodes, connect states with transition links, move and edit items in the workspace, and pan the view. A property panel at the right of the screen allows the user to edit details for a state node or a transition link. Pressing the Delete key deletes the selected item (if there is one) and any dependent items.

Realtor Filter

An app to support potential buyers by setting filters on houses for sale.

×

I coded a user interface to demonstrate my skills with basic JavaFX development (including widgets, event handling, layout, app functionality and the development of composite widgets). The UI is for a simple JavaFX app that allows the user to set filters in a list of houses for sale. The app provides controls for filtering based on price, number of bedrooms, or the area of the house in square feet. Filters provide range sliders which allow the user to set both a lower and an upper amount for the filter. The user can store the current set of filters as a search; when they do this, the search is added to a second list. Saved searches can be loaded by selecting them from the list and clicking a button. The app will help realtors quickly locate suitable homes to show to their clients.

More About Me

I am a User Experience Designer, Product Designer and Software Developer driven by a passion for problem-solving and a deep appreciation for beauty in all things. Travel has always been one of my greatest sources of inspiration, which led me to explore New Zealand and Australia for two exhilarating years. Since then, I have had the opportunity to explore different countries such as Japan, China, Costa Rica, and Mexico. Through these cultural experiences, I developed a keen eye for detail and an ability to understand people from various walks of life. As a result, my empathy towards human behaviour guides my design process towards solutions that truly resonate with users.

Outside of work, I channel my creative energy into photography, animation, printmaking and fabric dyeing. Not only do these mediums serve as stress relievers after long hours behind the computer screen, but they also allow me to think outside the box when crafting unique and compelling interfaces for the end user. Lastly, hiking, playing soccer, or simply getting my hands dirty gardening help me clear my mind and rejuvenate my spirits so that I can face another day full of challenges head-on. All in all, I believe design is a reflection of our own personality and values, therefore my goal is to create meaningful interactions that leave a lasting impact for people.
